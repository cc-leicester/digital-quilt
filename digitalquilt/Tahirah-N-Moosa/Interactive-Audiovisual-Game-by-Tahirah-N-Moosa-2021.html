Interactive Audiovisual Game by Tahirah N. Moosa

Interactive Audiovisual Game is an artwork realised by the student Tahirah N. Moosa in 2021, as part of the Year 1 Digital Arts Lab assessment.

Description: This is a mini-game where the user needs to get to the other side of the screen. The user has 5 seconds to enter the correct combination of keys as shown at the top of the screen.

How to intercat with the artwork: Press the correct keys using the numpad, sound is used within the artwork.
